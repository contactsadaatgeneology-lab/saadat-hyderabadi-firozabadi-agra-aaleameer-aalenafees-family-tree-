# saadat-hyderabadi-firozabadi-agra-aaleameer-aalenafees-family-tree-
Historical genealogy archive of the Naqvi Bukhari Sayyid families of Hyderabad Sindh, Firozabad, Agra, Lahore, Multan Uch Shareef and Karachi tracing lineage from Imam Ali al-Naqi (A.S) through Hazrat Jalaluddin Surkh Posh Bukhari (R.A) to modern descendants.
